# Module 8 - Describe Cost Management in Azure

Status: ✅ Completed

## Exam Objective

Understand pricing models, cost drivers, and cost optimization tools in Azure.

## Core Definitions

- Pay-as-you-go: Billing based on actual resource consumption.
- Reserved capacity/instances: Lower pricing in exchange for commitment term.
- Spot pricing: Discounted compute with potential eviction.
- Total Cost of Ownership (TCO): Full cost comparison between on-prem and cloud operations.
- Cost management: Monitoring, analyzing, and optimizing spend.
- Budget: Planned spending threshold with alerting.

## Major Cost Drivers

- Compute size and runtime duration.
- Storage volume, tier, and redundancy level.
- Data transfer (especially egress).
- Licensing model and marketplace services.
- Region-specific pricing differences.

## Cost Tools and Their Purpose

- Azure Pricing Calculator: Estimate projected costs before deployment.
- Azure TCO Calculator: Compare estimated cloud costs with on-prem model.
- Azure Cost Management + Billing: Track spend, create budgets, and analyze cost trends.

## Cost Optimization Techniques

- Right-size resources based on real usage.
- Use autoscaling and stop/deallocate non-production resources.
- Select appropriate storage tiers and lifecycle policies.
- Use reservations/savings plans for predictable workloads.
- Apply tagging for chargeback/showback and governance.

## Common Exam Traps

- Lower unit price is not always lower total cost if overprovisioned.
- Reserved options reduce cost only when utilization matches commitment.
